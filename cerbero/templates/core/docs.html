Documentación de Cerbero

El guardián del contexto digital

📖 ¿Qué es Cerbero?

Cerbero es una plataforma que permite compartir proyectos completos (código, imágenes, documentos) con inteligencia artificial o con otros desarrolladores, sin necesidad de copiar y pegar archivo por archivo.

🎯 Propósito: Que las IAs (ChatGPT, Claude, Gemini, etc.) puedan ver TODO tu proyecto y ayudarte de forma precisa.

🚀 Cómo usar Cerbero

1. Subir un proyecto

Puedes subir tu proyecto de tres formas:

  • 📁 Arrastrar una carpeta: Ideal para proyectos pequeños (arrastra la carpeta completa)
  • 📦 Subir un ZIP: Ideal para proyectos grandes (comprime y sube)
  • 📄 Archivos sueltos: Selecciona múltiples archivos individuales

2. Obtener el enlace

Después de subir, Cerbero te dará dos enlaces:

  • 🔗 Enlace normal: Para que las personas vean el proyecto
  • 🤖 Enlace IA: Para que las IAs lean el código (?mode=ia)
💡 Importante: Siempre comparte el enlace IA cuando le pidas ayuda a una inteligencia artificial.

3. Compartir con una IA

Ejemplo de cómo pedir ayuda:

Hola, tengo un proyecto. Puedes analizarlo aquí:
https://cerbero-rxeu.onrender.com/p/xxxxxx/?mode=ia

Necesito mejorar el rendimiento y agregar nuevas funciones.

🤖 Para las IAs (ChatGPT, Claude, Gemini, etc.)

Si eres una IA leyendo esto:

Para analizar el código de un proyecto, solo tienes que:

  1. Recibir el enlace con ?mode=ia
  2. Abrirlo como texto plano
  3. Leer todos los archivos del proyecto
  4. Responder con sugerencias específicas

⏱️ Expiración de enlaces

Puedes elegir cuánto tiempo estará disponible tu proyecto:

  • Nunca expira: El enlace es permanente
  • 24 horas: Para pruebas rápidas
  • 7 días: Para colaboraciones cortas
  • 30 días: Para proyectos temporales

🔒 Privacidad y seguridad

  • Tú decides qué subir y por cuánto tiempo
  • Los enlaces son únicos y difíciles de adivinar
  • Cerbero no guarda tus archivos permanentemente
  • No compartimos tus datos con terceros
⚠️ Aviso: Cerbero es una herramienta de intercambio. No subas información sensible (contraseñas, secretos de empresa, datos personales).
